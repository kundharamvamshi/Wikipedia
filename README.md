# Wikipedia Search App

A simple web app that lets users search Wikipedia and view matching results in a clean list.

## Features

- Search Wikipedia by typing a keyword and pressing `Enter`
- Show a loading spinner while results are being fetched
- Display result title, link, and short description
- Open results in a new tab

## Project Files

- `wikipedia.html` - page structure
- `wikipedia.css` - styles for the search page and results
- `wikipedia.js` - search logic and API handling

## How to Run

1. Download or clone this repository.
2. Open `wikipedia.html` in your browser.
3. Type a keyword and press `Enter`.

## Tech Used

- HTML
- CSS
- JavaScript
- Bootstrap 4
- Fetch API

## Notes

This project uses the `https://apis.ccbp.in/wiki-search` API to get Wikipedia search results.
